Dev Blog

The good'ole nostalgic mods

Trying to hit all the old mods we used to play with back in my HS days.

The Aether has been updated!

ChocoCraft has been updated!

Buildcraft does not have a modern equivalent but...

  • IC2 Classic has included pipes similar to BC which I favor over Pretty Pipes even though the latter looks more accurate
  • Additional Enchanted Miner essentially adds back the BC quarry
  • Auto-crafting is mostly handled in new ways by other mods better

Mo Creatures does not have a modern equivalent but...

  • Alexs Mobs is the closest comparable mod but doesn't fit this pack
  • Critters and Companions and Creatures and Beasts sorta fill out the world like Mo Creatures did
  • Ecologics
  • Ducklings

Industrial Craft is continued as IC2 Classic

Red Power is continued as Project Red

Forestry is continued as Forestry Community Edition

  • Bee breeding for dayz...

Railcraft is continued as Railcraft Reborn. I won't include.

  • There is a 1.19.2 beta version but it isn't maintained actively
  • I don't think anyone will really explore Railcraft
  • Added Steve's Carts to explore instead

Equivalent Exchange continued as Project E. I won't include.

  • I added some other magic mods that don't break game balance and seem more fun to checkout

Storage networks + Transport pipes

  • IC2 Classic includes Buildcraft like pipes
  • Project Red - Expansion includes old Red Power pipe networks
  • Pipez will provide an additional way to link up inventories just in case
  • Refined Storage is a powered storage network
  • Sophisticated Storage is a lower tech storage solution. Shares upgrades with Sophisticated Backpacks
